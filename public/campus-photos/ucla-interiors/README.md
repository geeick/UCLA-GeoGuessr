# UCLA interior photos

Put your real UCLA interior photos in this folder.

The filenames should match the `imageUrl` values in `src/data/pictureLocations.js`.
For example:

```txt
powell-library-main-reading-room.jpg
ackerman-union-food-court-area.jpg
broad-art-center-gallery-space.jpg
```

Photos are intentionally not included in this starter because many UCLA/interior photos online are copyrighted or require permission.
